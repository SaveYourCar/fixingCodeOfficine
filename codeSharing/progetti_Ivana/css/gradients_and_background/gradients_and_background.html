Linear Gradients

background: linear-gradient (color-top, color-bottom);

In this example, the colors blue and black are used to create a linear gradient from top to bottom.

background: -moz-linear-gradient (color-top, color-stop, color-bottom);

In this example, the linear gradient has multiple color stops and runs from top to bottom.

background:-moz-linear-gradient (color-top value%, color-stop value%, color-bottom value%);

In this example, the linear gradient has multiple color stops and their position is specified.

background: -moz-linear-gradient(left, color-top, color-stop, color-bottom);

In this example, the linear gradient has multiple color stops and they move from left to right.

background: -moz-linear-gradient(bottom, color-top, color-stop, color-bottom);

In this example, the linear gradient has multiple color stops and they move from the bottom to the top.

background: -moz-linear-gradient (value_deg, color-top, color-stop, color-bottom);

In this example, the linear gradient has an angle of 100deg.

background: -moz-repeating-linear-gradient (color-top, color-stop, color-bottom);

In this example, the linear gradient is repeated.

Radial Gradients

background: radial-gradient (position, shape or size, color-stops);

In this example, we didn't specify the shape of the radial gradient. So it will take the ellipse value by default.
